The Piclean Project - About

Cleaning up Picrew

What is Piclean?

Piclean is an open-source project aimed at improving the Picrew experience and archiving Picrew's data for preservation. The website you are currently on is the frontend for the Piclean API.

Why make it though?

Picrew is a fantastic platform for creating and viewing avatars and characters, but it has some limitations. Picrew has recently been introducing many intrusive ads on its platform. This is kind of scummy, considering the users who contribute content on Picrew don't get paid.

Additionally, Picrew has opted out of the Internet Archive's Wayback Machine, which means that if Picrew were to go down, all of its content would be lost forever. This is a significant concern, as Picrew hosts a vast amount of user-generated content that is not backed up anywhere else.

  • Piclean Frontend

    The frontend for interacting with the Piclean API. (this website)

  • Piclean API

    The API for accessing Picrew data. (backend for this website)

  • Piclean Scraper

    The scraper for collecting Picrew data into databases. Not public yet

Please follow Picrew's Terms of Service & the image maker author's licenses when using any scraped data/the image makers.
